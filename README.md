# Marvel Checklist

A simple and interactive Marvel Cinematic Universe (MCU) checklist designed to help fans track their viewing progress across movies, Disney+ series, specials, and sagas.

## Features

- Complete MCU watch checklist
- Track watched and unwatched titles
- Progress tracking
- Clean and responsive interface
- Runs entirely in the browser
- No installation required

- HTML
- CSS
- JavaScript

## Usage

1. Open the website.
2. Mark titles as watched.
3. Track your progress through the MCU.
